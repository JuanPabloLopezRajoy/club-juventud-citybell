# Club Juventud City Bell

## Descripción

Sitio web institucional desarrollado para el Club Juventud City Bell como proyecto académico de la materia **Técnicas de Maquetación e Interfaces Web**.

El objetivo del sitio es comunicar la identidad del club, sus actividades deportivas, noticias, formas de participación y canales de contacto mediante una interfaz clara, accesible y completamente responsiva.

El proyecto fue desarrollado aplicando principios de:

- HTML5 semántico
- CSS3
- CSS Grid
- Flexbox
- Container Queries
- Mobile First
- Design System
- Atomic Design
- Diseño Responsivo

---

## Organización

**Club Juventud City Bell**

Institución deportiva y social dedicada a la formación integral de niños, jóvenes y familias a través del deporte. El club promueve valores como el compañerismo, el respeto, el esfuerzo y el sentido de pertenencia, fortaleciendo el vínculo con la comunidad de City Bell.
